## Community Group Buying Mall System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0b73f57e2c8046c2854677e96fbf716c/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Product Information
5. Group Purchase Information
6. Community Information
7. Personal Center
8. Backend Management
9. Shopping Cart


**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center
4. User Management
5. Product Information Management
6. Product Type Management
7. Group Purchase Information Management
8. System Management
